Данное приложение является проектом для Яндекс.Лицея.
Данное приложение использует PyQt5.

В игру может играть до 4 человек. 
Можно менять уровень сложности.

Суть игры:
Компьютер загадывает слово и даёт подсказку к нему,
тот, кто отгадает слово побеждает!
Игроки по очереди выбирают буквы,
если игрок выбирает букву, которая присутствует в слове,
то он продолжает ходить, иначе ход предаётся 
следующему игроку в очереди.

Суммарно у игроков есть определённое попыток ошибиться,
их количество определяется сложностью игры.

Присутствует функция добавления слов в игру.
Загаданное слово должно содержать в себе 
только буквы русского алфавита.
Все слова, используемые в игре, хранятся в файле data.txt.

Good Luck!
