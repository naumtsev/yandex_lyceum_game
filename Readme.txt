Данное приложение является проектом для Яндекс.Лицея
Для использования приложения необходимо установить такие
библиотеки как: random, PyQt5, PyQt5-tools

В игру может играть до 4 человек. 
Можно менять уровень сложности

Суть игры:
Даётся загаданное слово и подсказка к нему.
Игроки по очереди выбирают буквы,
Если игрок выбирает букву, которая присутствует в слове,
то он продолжает выбирать следующую буквы,
иначе ход предаётся следующему игроку в очереди.
Суммарно у игроков есть определённое попыток ошибиться,
их количество определяется сложностью.

Так же присутствует функция добавления слов в игру.
Загаданное слово должно содержать в себе 
только буквы русского алфавита.

Good Luck!
